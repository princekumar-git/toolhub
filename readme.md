# ToolHub | All-in-One Digital Arsenal

**ToolHub** is a sleek, minimalist suite of essential web utilities designed for developers, creators, and daily productivity. Built with a focus on speed and aesthetic clarity, it provides a centralized hub for common tasks‚Äîfrom currency conversion to precise weather tracking‚Äîall within a single-page, responsive interface.

## üõ† Features

* **Weather Station**: Access real-time global weather data using city search or precise geolocation via the OpenWeather API.
* **JSON DevTools**: Quickly format, beautify, and minify JSON strings for debugging and development.
* **Currency Converter**: Perform live global exchange rate conversions using up-to-date financial data.
* **Clock & Timing Suite**: Integrated system clock, high-precision stopwatch, and a countdown timer.
* **Secure Password Generator**: Generate cryptographically strong, random keys with customizable complexity.
* **Calendar Suite**: A full-featured interactive calendar including an age calculation utility.

## üîß Installation & Setup

1. **Clone the repository:**
```bash
git clone https://github.com/your-username/toolhub.git

```


2. **API Configuration**:
* To use the weather features, obtain a free API key from [OpenWeatherMap](https://openweathermap.org/api).
* Open `index.html` and replace the `apiKey` placeholder with your active key:
```javascript
const apiKey = "YOUR_ACTUAL_API_KEY";

```




3. **Launch**: Simply open `index.html` in any modern web browser.
